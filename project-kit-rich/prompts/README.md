# Prompts

## EN
This folder stores reusable prompts for Codex and supporting planning notes for ChatGPT.

- `codex/` — step-by-step prompts for extraction, cleanup, and integration
- `chatgpt/` — project context, planning notes, architecture notes

Keep prompts short, direct, and task-specific.
