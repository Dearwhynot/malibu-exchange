# Project Context

## EN

Malibu Exchange is a small WordPress backoffice project for Telegram-bot-related exchange operations.

### Key ideas
- WordPress-based
- no React unless explicitly requested
- jQuery is acceptable and practical
- small team / small system / fast iterations
- multi-organization architecture from the start
- reusable modules preferred over one-off code
- visual source should come from the purchased HTML template

### Current long-term goal
Build a reusable modular starter system that can be quickly adapted to future backoffice projects.

## RU

Malibu Exchange — это небольшой WordPress backoffice-проект для операций, связанных с Telegram-ботом обменного сервиса.

### Ключевые идеи
- проект на WordPress
- без React, если он специально не нужен
- jQuery допустим и практичен
- маленькая команда / маленькая система / быстрые итерации
- multi-organization архитектура с самого начала
- переиспользуемые модули лучше, чем одноразовый код
- визуальный источник должен идти из купленного HTML-шаблона

### Текущая долгосрочная цель
Собрать переиспользуемую модульную starter-систему, которую можно быстро адаптировать для будущих backoffice-проектов.
