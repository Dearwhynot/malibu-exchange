# QA Template

## EN

Use this exact testing block after every completed integration step:

```text
ОБЯЗАТЕЛЬНОЕ ТЕСТИРОВАНИЕ (сделай сразу):
1) Где открыть
2) Что нажать (пошагово)
3) Ожидаемый результат
4) Что считается провалом
```

### Notes
- do not skip QA
- do not move to the next task until result is confirmed
- if the result is negative, fix immediately with a focused patch
